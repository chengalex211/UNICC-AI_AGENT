## 🚀 三步开始使用

### 第1步：安装依赖

```bash
# 安装Python包
pip install langchain openai chromadb tiktoken

# 或使用requirements.txt
pip install -r requirements.txt
```

### 第2步：设置API密钥

```bash
# 设置OpenAI API密钥
export OPENAI_API_KEY='your-api-key-here'

# 或在.env文件中设置
echo "OPENAI_API_KEY=your-api-key" > .env
```

### 第3步：运行RAG系统

```bash
# 首次运行：构建向量库（5-10分钟）
python quick_start_rag.py --setup

# 单次查询
python quick_start_rag.py --query "What are high-risk AI systems?"

# 交互模式（推荐）
python quick_start_rag.py --interactive
```

### Q: 构建向量库需要多久？

约5-10分钟，取决于：
- 文档大小（~850KB）
- 网络速度（调用OpenAI API）
- 电脑性能

只需运行一次，之后直接使用。

### Q: 向量库存储在哪里？

默认位置：`./vectorstore/`

可以自定义：
```bash
python quick_start_rag.py --vectorstore-dir "./my_vectorstore" --setup
```

### Q: 如何提高查询质量？

1. **更具体的问题**
   - ❌ "Tell me about AI"

2. **明确法规**
   - ❌ "What are the data protection rules?"
   - ✅ "What are the data protection requirements in GDPR Article 5?"

3. **使用法律术语**
   - ❌ "What can't AI do?"
   - ✅ "What AI practices are prohibited under the EU AI Act?"

## 📖 进阶使用

### 自定义Prompt

编辑 `quick_start_rag.py` 中的 `template` 变量：

```python
template = """你的自定义prompt..."""
```

### 调整检索数量

修改 `search_kwargs`:

```python
retriever = self.vectorstore.as_retriever(
    search_kwargs={"k": 10}  # 返回更多结果
)
```

### 使用不同的LLM模型

```python
llm = ChatOpenAI(
    model="gpt-4",  # 或 "gpt-3.5-turbo"
    temperature=0

```
